Carbuncle Game Engine

Welcome to Carbuncle

With this tutorial you will have the base understanding on how to build a game and play it using Carbuncle.

As of today, this version is lacking documentation and some features, but it's possible to build games using this.

Setting Up

Setting Up on Windows

Just download the binary file at the homepage and you are ready to go!

Setting Up on Mac OSX

On Mac the process is mostly the same, you will probably require to authorize it to run on your PC in the security tab for now.

Setting Up on Other platforms

At the moment, the rest of the platforms are not being built. In the future, ports to iOS, Android and Linux are planned.

Since Carbuncle 0.4 onwards, new methods can now be used for your application:

      
Using Carbuncle as a command line executable

      
You can now just do: $ /your/path/to/carbuncle /your/game/folder.

      
You can use this method to develop your game, but when you are done, you should use the distributable version below.

      
Launching code from the same folder

      
If Carbuncle reads a main.rb from the same folder it is currently installed, it will run that.

      
This is used to distribute your game with the carbuncle version you were using.

      
Note for MacOS users Because the Mac OSX version is an APP and not a binary file, your main.rb
        should be inside your mac.app/Contents/MacOS folder.
        You can always see the package's contents by right clicking and selecting "view package's contents"
